# Memória — Instruções de Manutenção

> Extraído de `banco-de-memoria.md`. Referência para manter as bases atualizadas corretamente.

---

## Quando atualizar os arquivos de memória?
- Após **cada implementação** aprovada pelo Auditor (Etapa 7)
- Quando uma **nova ADR** for tomada
- Quando um **problema** for descoberto ou resolvido
- Quando **pendências** forem concluídas ou criadas

## O que atualizar?
1. **Estado Atual** (`memoria/_core.md`): atualizar fase e próximo passo
2. **Implementações** (`memoria/_core.md`): adicionar novo registro IMP-XXX
3. **ADRs** (`memoria/adrs.md`): registrar se decisão arquitetural foi tomada
4. **Problemas** (`memoria/_core.md`): adicionar/resolver bugs
5. **Pendências** (`memoria/pendencias.md`): marcar concluídas ou adicionar novas
6. **MEMORY.md**: atualizar tabela de módulos se novo módulo criado
